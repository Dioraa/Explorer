# Explorer
Frontend Explorer Project

## Description  
**Explorer** is a frontend project designed as a travel agency website. It allows users to explore destinations like Dubai, view hotel options, and check prices. The website showcases various multimedia elements such as images, cards, and videos to provide a rich and engaging experience.

##Technologies Used  
- HTML  
- CSS  
- JavaScript
